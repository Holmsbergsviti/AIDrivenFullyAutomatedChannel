## 🖼️ How to Upload Avatar to Telegram

1. Open your channel in Telegram
2. Tap the channel name at top → ⚙️ Settings
3. Tap "Edit Profile Photo"
4. Choose `avatar.png` from this repository
5. Crop/adjust as needed
6. Save

---

## ✏️ How to Set Channel Description

1. Open channel → Settings
2. Tap "Edit Description"
3. Copy-paste the description above
4. Save
